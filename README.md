# TaskFlow - A Task Management App

### Live: https://dist-nine-jade.vercel.app

### Features of TaskFlow:

- User Authentication using Firebase and JWT
- User Private Route
- Add Task
- User Task Dashboard
- Update Task
- Delete Task
- To-Do List
- User Profile

### Technologies which have been used in this app:

- React
- React-Router
- Firebase
- Node.js
- Express.js
- MongoDB
